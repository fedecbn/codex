Notes: Habitats marins des départements d'outre-mer des Antilles. Correspond à la typologie "HABITATS_ANTILLES" de HABREF.


Préconisations :

  • Si on utilise le fichier ANTMER présent sur le site de l'INPN : on utilise le code CD_HAB, que l'on inclura dans l'attribut "codeHabitat", code métier de l'habitat, qui correspond au LB_CODE de HABREF.

  • Si on utilise HABREF, on prendra le CD_HAB directement, que l'on reportera dans l'attribut "codeHabRef".
