Présentation


Fonctionnalités

Pré-requis 
- Un serveur web
	-- ex. wampserver 2.5 (apache / php5 / extension php_pgsql.dll et php_pdo_pgsql.dll activées)
- Un système de gestion de base de données = Postgres (9.1 ou plus)
- git
	-- pour windows : https://git-scm.com/download/win
	-- pour mac : http://git-scm.com/download/mac
	-- pour unix : apt-get install git

Installation
1- clonez le depot git dans votre espace web
git clone ssh://[USER]@94.23.218.10:10022/home/gitsrc/codex.git [OUTPUT_PATH]
ex : git clone ssh://user@94.23.218.10:10022/home/gitsrc/codex.git C:/wamp/www/codex

2- Une fois le serveur web lancé, rendez-vous sur la page d'installation grâce à votre navigateur web
ex : localhost/codex

3- Suivez les instructions en renseignant les paramètres

4- vous pouvez vous connecter avec le login admin et mdp admin
